@delete projects

To delete all of the projects associated with your account, click the "delete" request button to initiate the request. You can only delete projects that you created through your account.
Warning: this operation is irreversible!

Authorization: Basic ***************
delete http://127.0.0.1:5000/projects/








@delete tickets

To delete all of the tickets associated with an individual project, click the "delete" request button after specifying the project you are requesting for. You can only delete tickets that you created through your account.
Warning: this operation is irreversible!

Authorization: Basic ***************
delete http://127.0.0.1:5000/projects/ /tickets





@delete project

To delete a single project, fill in the project id and then click the "delete" request button. You can only delete a project that you created through your account.
Warning: this operation is irreversible!

Authorization: Basic ***************
delete http://127.0.0.1:5000/projects/





@delete ticket

To delete a single ticket, fill in the project id and ticket id and then click the "delete" request button. You can only delete a ticket that you created through your account.
Warning: this operation is irreversible!

Authorization: Basic ***************
delete http://127.0.0.1:5000/projects/ /tickets/
