@post project

To post a project to your account, first fill out the JSON fields for the request body. Then, click the "post" button to initiate the request.

Content-Type: application/json
Authorization: Basic ***************
post http://127.0.0.1/projects/
{"project_name" : " ",
 "project_description" : " ",
 "project_category" : " "}







@post ticket

To post a ticket to a specific project, specify your project id, and then first fill out JSON fields for the request body. Then, click the "post" button to initiate the request.

Content-Type: application/json
Authorization: Basic ***************
post http://127.0.0.1/projects/ /tickets

{"ticket_name" : " ",
 "ticket_description" : " "}
